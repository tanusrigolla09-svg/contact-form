# Contact Form with JS Validation

A responsive contact form with client-side validation built using HTML, CSS, and JavaScript.

## Features
- Validates Name, Email, and Message fields
- Regex email validation
- Inline error messages with shake animation
- Success screen on valid submission
- Progress bar, character counter, live re-validation

## Structure
```
contact-form/
├── index.html
├── css/style.css
├── js/validate.js
└── js/form.js
```

## How to Run
Open `index.html` in any browser. No setup needed.

## Tech Used
HTML · CSS · Vanilla JavaScript
